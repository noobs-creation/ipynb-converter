# IPYNB-Viewer

View IPYNB (Jupyter Notebook) files without having to fire up Anaconda or Jupyter, just view your file directly in your browser.

## Setting Up
Install the dependencies
```bash
pip install -r requirements.txt
```

## Usage
Viewing an IPYNB file
```bash
python ipynb-viewer.py mynotebook.ipynb
```
Getting Help
```bash
python ipynb-viewer.py -h
python ipynb-viewer.py --help
```
